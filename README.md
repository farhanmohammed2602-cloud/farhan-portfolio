## 🖼️ How to Add Your Profile Photo

1. Place your photo in the `assets/` folder and name it `profile.jpg`
2. Open `index.html` and find this comment block (around line 235):

```html
<!--
  TO ADD YOUR PHOTO:
  1. Place your image in the assets/ folder (e.g. assets/profile.jpg)
  2. Remove the placeholder div below
  3. Uncomment this line:
  <img src="assets/profile.jpg" .../>
-->
<div class="profile-placeholder">MFA</div>
```

3. Remove the `<div class="profile-placeholder">` and the sub-div
4. Uncomment the `<img>` tag

---

## 🚀 Deploy to GitHub Pages

1. Push this folder to a GitHub repository
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at: `https://yourusername.github.io/farhan-portfolio`
